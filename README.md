## 🚀 Getting Started

### Prerequisites

- Node.js 22+
- pnpm
- Rust
- Cargo

### Clone

```bash
git clone https://github.com/<your-username>/Aria.git
cd Aria
```

### Install

```bash
pnpm install
```

### Development

```bash
pnpm tauri dev
```

### Production Build

```bash
pnpm tauri build
```

Build output is generated in:

```
src-tauri/target/release/bundle
```
